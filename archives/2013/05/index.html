各种奇葩的一天

吃饭记

在上海科技馆地铁站附近的一家小店解决午饭，这家店的服务员屡次爆出奇怪的语录，让我和man姐其囧无比

没开水

我问：可以给我倒杯白开水吗？

服务员答：我们平时是都有提供的，但是今天是周末，客人太多了，我们没空烧开水

我：。。。。

喜欢吃鸡

我点菜：来份三黄鸡吧

服务员：最近没有这份菜

我问：是不是禽流感，鸡类都没有啊？

服务员：不是的，三黄鸡是凉菜，热菜的鸡类还是有的。

我：那来份你这里的招牌xx鸡吧

服务员跑去厨房问了下，回来说没有。

我：这不是热菜吗？而且还是招牌菜，怎么就没有啊？

服务员：呃。。。你们就那么喜欢吃鸡吗？

我：。。。。

我：那好吧，我们点啤酒鸭可以了吧。

我翻过菜牌看到冬菇蒸鸡，跟man姐用粤语说了句，这应该也没有吧。

服务员：这个是有的，肯定是有的。

我和man姐：。。。。。。

买鞋遇黄牛

下午去逛了下五角场，Asics正好在打折，然后想买双鞋，然后第一次遇上黄牛。没想到买鞋也有黄牛~~~~

商场搞活动，买200送200券，当然要是买鞋，当然想直接打5折，要券干嘛。这时黄牛就出现了，到处协调赚中间价。比如我买鞋，黄牛帮我用正价买单，赠券归黄牛，然后我按五五折付给黄牛现金。

为我买单的黄牛果然牛。背着个小包，整个一个傻根的样子。买完单，直接把发票和鞋给我。我问他哪里有ATM可以取现金，然后他说他带我去。

一路上，他一直走前面，根本没有看我们，而我们是拿着鞋和发票，而且我一分钱都还没付。。。

他一下子走很快，而且他一下子走一个方向，我们几乎要走很快，在人群中不断找，才能跟上他。我们当时一直在想，要是我们突然走掉他就惨了。走到电梯前，他一下子按上，一下子按下，我问他是在楼上还是楼下，他说楼上，然后带着我们坐一部下行的电梯，是在很无语。

走到银行，他在外面等，然后我们进去取钱。银行有两个门，而且人超多的。我偷偷看了一眼，他居然背着门在抽烟，我们是在担心他的下一个顾客有没有想我们这么老实。等我取好钱给他，他居然简单数了一下，根本没看是不是真的，就直接快步走开消失在人群中，留下我和man在发愣，我们只好默默祝愿他，好人一生平安。

Sublime Markdown组合成轻量级笔记

Sublime

之前的编辑器一直在使用NotePad++,虽说没什么太多使用的功能，但是觉得用得习惯，也就一直坚持用了下来。最近经Z总介绍，发现了Sublime这个编辑器，试用了大概一周，果断换代NotePad++，改用Sublime。

关于Sublime的简单介绍可以参考Sublime Text 2 入门及技巧

简单来说，吸引我的地方有：

  • 很炫的界面
  • 很好的扩展，插件很多，非常实用
  • 大部分操作都可以使用命令的形式输入

Markdown

Markdown是一种轻量级标记语言，开源社区用得很广。感觉不错是因为它能提供一个简单的笔记排版功能，相对于Word鼠标操作十分繁琐来说，配合Sublime和Vim来用，可以说几乎不用点鼠标，更加专注在文字上，而不是排版上。

当然，Markdown并不是所写即所见的（和LATEX一样），所以一开始会有点不习惯。可是使用一段时间后，觉得很顺手，甚至觉得有点不够用，想去用LATEX。所以感觉LATEX上手很难的话，可以先使用一下Markdown，体验一下这种标记语言的魅力也不错。

Markdown的语法很简单，感兴趣的话，可以看看Markdown 语法说明 (简体中文版)

Sublime+Markdown

要想把Markdown的内容显示出来的，需要在Sublime中装插件。很多人会推荐Markdown Previewer和MarkdownEditing，我试用后觉得都太弱了。如果用这个组合的话，强烈建议使用OmniMakeup Previewer这个插件。

相比于Markdown Previewer，OmniMakeup Previewer不仅能够实时渲染，而且支持多种格式混合使用，比如可以在Markdown中使用LaTeX的数学公式。

比如需要显示 a2+b2=c2 这样的公式，只需要输入$a^2 + b^2 = c^2$就可以了，使用了MathJax来渲染，支持LaTeX支持的所有数学公式

Sublime + Evernot

平时使用Evernote来做笔记，只要在Sublime中装上插件，可以把Markdown的文字渲染好直接发送到Evernote，这个功能真的满不错。只不过是渲染的效果很搓而已。如果需要把Markdown的内容写到笔记去，推荐两个方法：

使用OmniMakeup Previewer渲染，然后用Evernote Clip到Evernote笔记
改用Wiz笔记，最近在使用，感觉很不错

痛苦只是痛苦而已

有人说，今日的痛苦是明日的财富，可是我知道，痛苦只是痛苦而已，痛苦过后的思考才是真正的人生财富。

1.别人的话

黄小琥在《没那么简单》中唱道“别人说的话，随便听一听，自己做决定”。在心情郁闷的时候总喜欢找人聊一聊，然后很认真地倾听别人的话，试图从中得到领悟。现在终于发现，别人的话，真的听听就好。不是说别人的话不重要或者没有道理，可是那毕竟只是别人的想法，别人的人生感悟，或许有共鸣，或许有同感，可是这个世界上没有感同身受这回事，自己的痛苦永远只有自己知道，如果自己没能释怀，别人说得再多，对你来说也只是徒劳。尽管如此，依然需要感谢这些“徒劳无功”的“别人的话”，因为它给予力量去坚强。

2.苦恼

有人说，“用十年后的自己站在现在的角度看待现在的事情，你就会发现，也许十年后的自己根本就不在乎这些事情”，的确十年后的你根本不会在乎现在的苦恼，可是苦恼和痛苦是属于现在的你，你拥有十年后你的所没有的苦恼和痛苦，这又何尝不是一种乐趣。预期去思考十年后的自己该如何看待当前的痛苦，还不如好好享受现在的苦恼，或许十年后的你会怀念这样的苦恼。

3.无奈

有些东西你无法放下，并不代表你必须拥有，这只是一种无奈。人最大的苦恼正是来自于无奈。对于无奈的事情，除了继续无奈，还是只能无奈。终于明白，有些事不是你努力去做，认真去做，就一定可以得到。相反有时候越是挣扎，越是感到沮丧，越是没办法做到。与其在无奈中挣扎，还不如承认这只是一种非接受不可的无奈。

4.伤感

在思南路上闲逛，在光怪陆离的街道上感受车水马龙，忽然之间有种伤感。这种淡淡伤感并不是《一路向西》中经过极大欢乐后的惆怅，而是无法融入这场盛大的欢乐而产生的伤感。魔都，虽然她有令人向往的小资，有大都市的繁华，可对我来说，依然是一座陌生的城市，正如汇集所有珍贵食材的美食摆在你面前，你发现突然间没有了想吃的欲望，傻傻地看着眼前的美食，却还在回味上一道菜。这种伤感令人对现实产生怀疑。甚至很想逃离。

5.生活

每天的生活很简单，工作，阅读，跑步，睡觉。阅读给我平静，而跑步给我宁静。偶尔需要一点点酒精帮助睡眠。
