Hexo3 建立 GitHub Page

  • Skills

前一阵时间在忙毕业论文的事情，很久都没有更新博客了。最近想着更新一下博客，却遇到一系列的问题，怎么也没有办法发布新的博文，最后经过一番捣鼓，终于能够重新发布。由于重装过电脑，把软件什么都删除了，所以这次需要重新弄，这篇博文把整个过程记录下来，以防以后遇到相同的问题可以查阅。

Git

GitHub Page使用Git来管理，因此第一步是安装Git. 在Windows下安装Git有三种方法：

  • Cygwin
  • Github for Windows
  • Git

我的电脑里本来就有Cygwin，因此我就直接在Cygwin上安装。

SSH-Key

生成 ssh-key

安装完Git之后，首先要做的是配置SSH，用于让本地Git项目与远程的GitHub建立联系。

重新生成新的ssh key

$ ls ~/.ssh
$ rm -rf ~/.ssh/*
$ ssh-keygen -t rsa -C "johnnyluck@yourmail.com"
Enter file in which to save the key:(输入key的名称，或者直接回车跳过)
Enter passphrase:
Enter same passphrase again:

这样就可以生成ssh key

添加ssh key 到GitHub

这一步很简单，在GitHub页面上， Account Settings—->SSH Public keys —-> add another public keys

将id_rsa.pub的内容粘贴到Key的框内即可

开启 ssh agent

这一步总算把我坑惨了，之前重来没有发现有这个问题，在hexo deploy的时候一直说是提示错误 ： Permission denied (publickey)

具体解决方法可以参考GitHub的帮助文档Error: Permission denied (publickey)

$ eval $(ssh-agent -s)
$ ssh-add ~/.ssh/id_rsa
$ ssh-add -l

测试

$ ssh -vT git@github.com

输入yes之后，如果反馈是如下，则说明配置key成功了

You've successfully authenticated ...

Hexo 3.0

由于之前一直使用的是Hexo 2.x，现在Hexo已经更新到3.0版本，因此如果是使用Hexo3.0，那么需要做一些相应的改动（之前网络上的脚本大多都是以Hexo 2.x为例子的）

安装Node.js

Node.js官网下载安装包，并安装。

按照Node.js之后，在Cygwin内即可直接使用，如果无法使用估计是系统变量path没有设置对，只有简单设置就好

$ npm --version

可以用来查看Node.js是否安装成功

安装Hexo3.0

$ npm install hexo

建立博客文件夹，初始化hexo. 

$ mkdir johnny
$ cd johnny
$ hexo init

安装hexo对应的插件，这一步非常重要，这也是Hexo3.0和之前版本比较大的区别

$ npm install
$ npm install hexo-generator-index --save
$ npm install hexo-generator-archive --save
$ npm install hexo-generator-category --save
$ npm install hexo-generator-tag --save
$ npm install hexo-generator-feed@1 --save
$ npm install hexo-generator-sitemap@1 --save
$ npm install hexo-server --save
$ npm install hexo-deployer-git --save
$ npm install hexo-renderer-marked@0.2 --save
$ npm install hexo-renderer-stylus@0.2 --save

发布

修改_config.yml文件，这里Hexo3.0和之前的版本有一点不一样:

deploy:
  type: git 
  repo: git@github.com:johnnyluck/johnnyluck.github.io.git
  branch: master

deploy的type之前一直是github，现在改成是git，其他的基本上都是一样的。

至此，大致流程已经完成，配合网络上的教程，应该能够轻松完成建立GitHub Page的操作了。
