Sublime Markdown组合成轻量级笔记

  • Skills
Tech

Sublime

之前的编辑器一直在使用NotePad++,虽说没什么太多使用的功能，但是觉得用得习惯，也就一直坚持用了下来。最近经Z总介绍，发现了Sublime这个编辑器，试用了大概一周，果断换代NotePad++，改用Sublime。

关于Sublime的简单介绍可以参考Sublime Text 2 入门及技巧

简单来说，吸引我的地方有：

  • 很炫的界面
  • 很好的扩展，插件很多，非常实用
  • 大部分操作都可以使用命令的形式输入

Markdown

Markdown是一种轻量级标记语言，开源社区用得很广。感觉不错是因为它能提供一个简单的笔记排版功能，相对于Word鼠标操作十分繁琐来说，配合Sublime和Vim来用，可以说几乎不用点鼠标，更加专注在文字上，而不是排版上。

当然，Markdown并不是所写即所见的（和LATEX一样），所以一开始会有点不习惯。可是使用一段时间后，觉得很顺手，甚至觉得有点不够用，想去用LATEX。所以感觉LATEX上手很难的话，可以先使用一下Markdown，体验一下这种标记语言的魅力也不错。

Markdown的语法很简单，感兴趣的话，可以看看Markdown 语法说明 (简体中文版)

Sublime+Markdown

要想把Markdown的内容显示出来的，需要在Sublime中装插件。很多人会推荐Markdown Previewer和MarkdownEditing，我试用后觉得都太弱了。如果用这个组合的话，强烈建议使用OmniMakeup Previewer这个插件。

相比于Markdown Previewer，OmniMakeup Previewer不仅能够实时渲染，而且支持多种格式混合使用，比如可以在Markdown中使用LaTeX的数学公式。

比如需要显示 a2+b2=c2 这样的公式，只需要输入$a^2 + b^2 = c^2$就可以了，使用了MathJax来渲染，支持LaTeX支持的所有数学公式

Sublime + Evernot

平时使用Evernote来做笔记，只要在Sublime中装上插件，可以把Markdown的文字渲染好直接发送到Evernote，这个功能真的满不错。只不过是渲染的效果很搓而已。如果需要把Markdown的内容写到笔记去，推荐两个方法：

使用OmniMakeup Previewer渲染，然后用Evernote Clip到Evernote笔记
改用Wiz笔记，最近在使用，感觉很不错
