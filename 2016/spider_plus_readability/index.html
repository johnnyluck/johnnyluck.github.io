当网络爬虫遇上Readability

  • python Skills
Tech

在Google Reader停止服务之前，我一直有使用RSS订阅博客的习惯。自从Google宣布关闭Reader之后，虽然也有一些替代应用可以使用，如国外的Feedly就提供了与Google Reader的无缝连接，而在国内则涌现出如鲜果和网易云阅读等服务。不知道是怎么不习惯还是RSS的质量越来越差了，后来我没有再使用这些应用。
最近由于工作上需要查阅文档，发现一大批相关的博客中有大量的技术干货，于是有了再次订阅RSS文章的想法，但这次，吸取之前的经验，不想订阅大量RSS，一天推送一大堆文章，导致根本没有时间去看，久而久之就没有兴趣。想到“一个”这个应用，因此在想，为什么不写一个程序，然后自动爬取文章，然后每天只推送一篇文章。

网络爬虫

要完成上面的想法，需要根据我的关键词对感兴趣的博客或者网站的文章进行抓取，因此这里需要一只网络爬虫。

Requests

要完成网络爬虫，HTTP的操作是基础，这里我选择了Requests这个“给人类使用的HTTP库”。对于Requests的安装可以通过pip和easy_install来实现。HTTP的请求类型包括PUT, DELETE, GET和POST，这些操作在requests中都非常容易实现。

request类

class requests.Request(method, url, headers, files, data, params…)

  • method: HTTP method, 如PUT,GET,DELETE和POST等.
  • url: 请求的URL
  • headers: HTTP的头部，使用字典来传递
  • data: HTTP的数据，使用字典来传递
  • params: URL参数，使用字典来传递

response 类

class response

  • apparent_encoding
  • content: content of the response, in bytes
  • text: content of the response, in unicode
  • raw: file-like object representation of response
  • json(kwargs): json-encoded content of response
  • iter_content()
  • iter_lines()
  • link:parsed header links
  • encoding:
  • headers:
  • status_code:Integer Code of response

session 类

class session

  • auth
  • cert
  • close()
  • delete()
  • get(url,kwargs)
  • head(url,kwargs)
  • post(url,kwargs)
  • put(url,kwargs)
  • send(request,kwargs): send a given PreparedRequest

简单的Get请求

import requests
res = requests.get(‘url’,params={‘key’:’value’}
resContent = res.content
resText = res.text

Beautiful Soup

在获得网页内容之后，我们需要对html进行解析与处理。这里使用beautifulSoup这个优秀的库来实现。bs能够非常快捷方便地定位html中的元素。

apt-get install Python-bs4
pip install beautifulsoup4

BeautifulSoup将html文档转换层一个复杂的树型结构，每个节点都是一个python对象，所有的对象可以分为tag, NavigableString, BeautifulSoup 和Comment。

关于BeautifulSoup的操作，可以参阅官方文档

requests + beautifulsoup4

使用request与beautifulsoup4结合，可以完成网络爬虫的各种操作

res = requests.get(‘url’)
soup = BeautifulSoup(res.text)
print(soup.title.name)

Readability

Readability 提供了python的API接口，使用起来也非常方便。下载地址

Auth

首先需要注册Readabiliy, 然后在account页中申请得到应用key和secret。(注意, 在网页中得到的key和secret在文档中称为consumer_key和consumer_secret)

在得到应用key之后，还需要使用账户和密码申请xauth得到的token_key和token_secret

from readability import ReaderClient

(token_key,token_secret) = auth,xauth(consumer_key=’consumerKey’ consumer_secret=’consumerSecret’, username=’yourname’, password=’yourpassword’)

以上代码只需要执行一次就能获得token，这时候在readability的网站上就能看到应用已经被激活。以后只需要使用这个token的key和secret就能操作了。

ReaderClient

from readability import ReaderClient
client = ReaderClient(token_key=’token_key’, token_secret=’token_secret’)

Readability 的API提供了阅读列表基本的操作：

add_bookmark(url, favorite=False, archive=False, allow_duplicates=True)
