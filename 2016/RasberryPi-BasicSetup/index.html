RaspberryPi 基本设置

  • RaspberryPi
Tech

系统安装

树莓派本质上只是一块焊好CPU的主板，要想启动树莓派还需要为其装上操作系统，为了顺利完成系统的安装，需要以下设备：

  1. 路由器
  2. SD卡
  3. 电脑

系统安装主要有两种方式，一种是直接刻录，另外一种则是利用树莓派基金会开发的工具NOOBS. 对于刻录安装这种安装方式， 就是将选择系统镜像，将其下载后刻录到SD上；而是用NOOBS则是进入引导模式，根据用户的选择安装，因此离线版则是把所有的系统都下载好，而在线版则是根据选择再下载。

因为我没有外接显示器，鼠标和键盘，所以我选择了直接刻录安装的方式。

系统镜像文件可以在树莓派官网下载，同时可以参考官网的安装参考流程。

SSH 连接

将刻录完的SD卡插入树莓派的卡槽，连接上网线，在接通电源后，树莓派便开始启动了。如果没有外接显示器，这时候根本无法获知树莓派的状态，不过可以观察一下其LED灯与网口LED灯的状态。如果系统成功启动的话，路由器上将显示树莓派接入了局域网，这时候就可以使用SSH连接了。

Username: pi
Password: raspberry

快速设置

在SSH登录树莓派后，可以通过命令快速设置树莓派。

sudo raspi-config

我的SD卡超过4G，因此选择了Expand Filesystem.

设置软件安装源

在系统默认是软件安装源的服务器在外国，因此安装软件会非常慢，因此需要更改安装源以便更好地加速。国内好一家高校已经有了树莓派的镜像：

  • 浙江大学
  • 重庆大学
  • 华中科技大学

根据提示更改/etc/apt/sources.list即可。

图形化界面

使用Putty SSH登录树莓派很多操作实在是不够方便，因此需要设置图像化界面。

VNC

sudo apt-get install tightvncserver

Windows 远程桌面

sudo apt-get install xrdp

挂载网络硬盘

由于家里的WDMyCloud需要使用，因此需要挂载在树莓派上。

sudo mkdir /mnt/nfs
sudo mount -t nfs -o nolock -o tcp 10.0.x.x:/nfs  /mnt/WdMyCloud

如果需要下次启动后自动挂载，编辑/etc/fstab，添加以下代码

:10.0.x.x/mnt/WdMyCloud /mnt/WdmyCloud nfs\_netdev, default,noatime,intr,netdev,user,auto 0 0

VPN 连接

pptp 客户端

sudo apt-get install pptp-linux

创建连接

sudo pptpsetup -create *VPN_Name* --server *Host Server Address*　\
--username *UserName* --password *Passwd* --encrypt

连接VPN

sudo pop *VPN_Name*

断开VPN

sudo poff

修改DNS

sudo echo "8.8.8.8" > /etc/resolv.conf

自动化脚本

#!/bin/bash
i=0
while [ $i -lt 5 ]
do 
  echo "PPTP connected to ytvpn start."
  pon ytvpn
  echo "Waiting connection ..."
  sleep 3
  wait
  rc=`ifconfig | grep ppp0`
  if [ $? -eq  0 ] ; then
    route del default
    route add default dev ppp0
    echo "# Generated by Johnny to use vpn" > /etc/resolv.conf
    echo "nameserver 114.114.114.114" >> /etc/resolv.conf
    echo "nameserver 8.8.8.8" >> /etc/resolv.conf
    echo "nameserver 8.8.4.4" >> /etc/resolv.conf
    echo "Connection completed successfully"
    exit
  else
    echo "Connection failed. Try latter..."
    sleep 5
    wait
    let i=i+1
  fi
done
