生命无常，珍惜眼前

昨晚在和爸爸商量工作上的问题的时候，爸爸突然跟我说，他的一个老友前几天出车祸走了。爸爸说得十分简短又异常的平静，以至于我一时难以置信，顿时哑口无言。

爸爸接着跟我说，考虑工作不需要太纠结，薪酬和工作内容其实也没有想象中的那么重要，反而最重要的是在工作中能够有所收获，和领导同事的相处能够愉悦，每天过得开心就好。

爸爸的老友，暂且称呼他作B伯伯，就是爸爸的工作伙伴，他们在同一家公司工作了三十多年，同时也培育了三十多年的友谊。他们每每遇到生活或者工作上的问题，都会跑出来喝酒聊一聊，两个人积累下来的默契我们身边的人是难以理解的。

记得印象最深的一次是爸爸几年前的一次小升迁，那时爸爸请客吃饭，爸爸的同事都过来了，那时在大家都喝了蛮多的时候，趁着爸爸出去结账，B伯伯当着大家的面说，他为爸爸的升迁感到十分高兴，他觉得这是爸爸应得的，希望大家以后对爸爸在工作上有什么意见和不满，一定和他说，或者冲着他去。

B伯伯对我也很好，每次和爸爸的同事出去吃饭的时候，在大家在喝得很high的时候，他总会关心我是否吃得饱。有好吃的菜上座的时候，总会先夹一块到我的碗里。在我读本科的时候，每次放假回家和他一起吃饭，他也给我塞一两百块，让我去买好吃的。

真的没有想到，生命的反转可以来得如此的悄无声息，十一国庆回家，我才跟着爸爸和他的老友一起喝酒，当时还喝得很畅快，现在突然就听到噩耗，以后再也没办法和他喝酒了。

我不知道爸爸的平静背后积淀了多少的悲伤，一个深交了三十多年的朋友，说走就走了。 我不知道爸爸以后在酒桌上是否还会惦记着他，也不知道爸爸在遇到生活的不如意的时候，是否想找人出来喝两杯。如果是我的话，我一定会很难过。

很多时候我总在纠结生活的烦恼，殊不知生命之无常。努力地过好当下的每一天，珍惜眼前人。

B伯伯，一路走好！

错综复杂的狗血八卦情感问题

我也想不明白我是怎么啦，放着Paper没心情写，居然跑来写博客，真是好颓废啊。最近一段时间，听说了身边的小伙伴的感情故事，真是各种狗血各种纠结，实在是看不透看不懂，我只能说我弱爆了。

淡定二人组

两个人在一群无聊人的起哄中发展出暧昧感情，女孩子动了感情而男孩子还“二”得很单纯，结果一场轩然大波的误会掀开序幕。好事之徒譬如我这种渣男在一旁推波助澜，害得男孩子更加迷茫而女孩子更加着急。不过不幸中的万幸，最终两人中其中一方捅破隔阂，轰轰烈烈的情感倾诉戛然而止。

不过令我等众人无法参透的，乃是二人关系依旧如故，似乎更加亲密。即使在众人面前仍然我行我素，谈笑风生，毫无避忌而聊得甚欢，屌丝如我在一旁目瞪口呆，真是大开眼界。尝旁敲侧击，试图了解两人关系到底是什么情况，两个都说大叫已经讲清楚，Move On了，现在是完全没有问题，100%的好朋友。

原谅我的情商，我依然没办法理解，既然曾经有过的感情，怎么可以说放下就可以完全放下，还是可以表现得如此淡定。想一想，心里难免一阵凉风吹过。要是这种情况放在自己的身上，我真想借用甄嬛传中的名句，“皇上，臣妾我做不到”。我实在做不到对着曾经有个感觉的那个人做100%好朋友，我宁愿两人形同陌路，正如她说的，“那些故事就埋在两个人的心里就好，这只是你们两个人的独家记忆”。

纠结三人游

帅哥DB同时被两个美女B和D追。美女B采用每天真人上门报道，而美女D采用微信聊天攻略，帅哥DB每天可以说是应接不暇。帅哥DB本来喜欢的是美女B，而当时美女B则是有男朋友，结果只能郁郁而放手。现在美女B单身了，并且变身主动，帅哥DB却说人家太花心，需要时间等待考察。美女D不巧正是美女B的好朋友，经美女B介绍认识帅哥DB，从此一发不可收拾喜欢上了男生DB，终于“一人留，两人疚，三人游”，狗血而刺激。

帅哥DB一直不知道如何选择，每天向我们说他有多烦恼，我在一旁听过之后咬牙切齿，各种羡慕嫉妒恨！！！对于我这种人来说，有个女神投怀送抱，赶紧跪下舔还来不及，怎么还不知道怎么选择呀，还要还是两个女神级的美女。B有B的好，D有D的好，B有B的问题，D也有D的问题，其实要说不知道怎么选择，也是因为可以选择得实在是太多了。问题也许并不是在别人身上，和谁在一起都没有安全感，也给不了别人安全感，这到底还是自己的问题呀。若是遇上对的人，这个对的人就是唯一的，是不需要选择的。若是没遇上对的人，怎么选到最后还是会选错。

超明显的地下情

女生刚分手，男生各种献殷勤，女生还来者不拒，最关键是这些殷勤还要是当着所有人的面非常明显地送出去。然后别人问起还假装什么事情都没发生过，若无其事。你们当所有人都瞎了眼么，还是把其他人都当傻X呀。男生呀，得不到同性的尊重，怎么赢得到异性的青睐呢？有些事情虽说是两个人的事，可以发生在一个实验室内部，还是不能太狗血~

这些天听着这些八卦，真的是颠覆我三观呀。才发现，原来我的情感是那么的单纯，实在无法理解其中的荡气回肠，惟有“我已经老了，感觉不会再爱了”之感悟。

碎片化信息时代的讯息阅读

互联网的飞速发展让我们现在每天接收的讯息爆炸性增加。微博，微信，新闻客户端，这些信息的发布源每时每刻推送大量的信息过来，这些碎片化的讯息让我们似乎知道很多，但同时也让我们花费更多的时间在无用的讯息上。时代飞速的步伐迫使我们重新思考，该如何更加有效地处理这些碎片化的信息。

Fenng在其的小道消息博客上曾经探讨过这个问题，《碎片化时间与我的阅读习惯》这篇博客介绍了Fenng利用碎片化时间进行阅读的习惯，其特点是利用一些软件或者应用来提高讯息阅读的效率。在我自己的碎片化阅读中也有类似的经验，在这篇博客中，简单地分享我的阅读习惯。

1.微博
碎片化时代的最重要产品就是微博，微博的火爆使得我们的讯息接受变得越来越碎片。但是最近很多人都发现，微博变得越来越“不好玩”了。微博上充斥着大量的营销微博，同时自从阿里巴巴入股新浪微博之后，微博的广告变得越来越多。

简单观察一下可以发现，微博的主要内容来源大概是几个方面：朋友的心情和内容分享，相关行业的讯息，新闻事件，草根大号的洗脑式教条和大V的人生哲学。对于我个人而言，微博内容的重要程度依照以上顺序逐渐下降的。所以利用好微博的分组功能，能够很好地过滤掉一些没有用的讯息。打开微博的时候，一般都是简单扫一遍朋友们的心情和内容分享，然后按照时间的优先，扫一遍新闻和行业讯息，之后就可以关掉微博了。

但是有时候会发现朋友们分享的内容也是良莠不齐，其中很多都是转发一些没营养的草根大号的洗脑式教条，这时候该怎么办呢？这时候就要利用chrome的一个插件，“眼不见心不烦”，通过关键词和黑名单过滤，比如”抽奖”，”营销“之类的关键词统统过滤掉。同时这个插件能够屏蔽掉网页中几乎所有的广告，也能自定义页面中的链接。

2.微信
微信中我也曾经试过订阅自媒体的信息推送，有些信息源每天推送大量的信息，往往都是来不及阅读，然后又有个小红点，对于有强迫症的人来说非常不爽。尽管这种情况在微信5.0以后有了大量的改善，但是我还是决定在微信中不订阅任何的自媒体推送信息，因为我的信息订阅有专门的应用在负责。

3.信息订阅
使用zaker和网易云阅读来订阅信息是我每天大量使用的功能，通过针对性地订阅信息，利用别人来为我进行信息筛选。能够更加有效地阅读我感兴趣的话题。在这个资讯爆炸的时代，资讯已经不再重要了，重要的是选择后的资讯。

喷嚏图褂，选取的是每日微博热门话题，每日只需要阅读一遍就可以知道每天微博上热烈讨论的话题。这也是我能够不看微博，却能够知道每天微博在热烈讨论着什么。

知乎每日精选，有人曾经说自己对一些话题的认识不够深入，这时候就应该订阅这个信息源。知乎上就是一群老妖，他们对一些领域的精通使得他们的回答能够深刻而独到。

cnBeta，IT资讯的集合

36kr，IT资讯和评论，大概几天会看一次。同时和爱范儿中的评论会用pocket抓下来，每周抽时间过一遍，加深对行业的理解。

4.pocket和花瓣
在互联网上有很多的知识或者图片，看过之后觉得很有用，但是时间又很赶来不及仔细看，这时候就需要使用稍后阅读的功能。pocket提供了收藏，稍后阅读的功能，同时其只能排版功能能够去除不必要的信息，让人更加关注在内容阅读上。zaker也有转发到pocket的功能，所有每天看到有深度的内容，就会转发到pocket，然后每周过一遍长文。

花瓣网则是提供收集的功能。当看到一些内容，觉得很有趣，以后可能会用到的时候，我就会转发到花瓣保存起来，有时间的时候整理一些里面的内容，有时候突然需要用到就可以迅速找到。

5.音频订阅
除了之前提到过的文字订阅之外，还有一种音频订阅功能，podcast！以前很多人会利用podcast来听订阅一些英语新闻收听，在获知新闻的同时也提高听力。我除了有时候听一下英文之后，更多地时候就是听一些有营养的电视节目。因为平时很少能接触到电视，所有很多电视节目的资讯没办法获取，而利用这个办法能够弥补这方面的缺失。我觉得凤凰的一些深度评论和访谈还是挺有用的。

6.Google订阅和RSS订阅
google订阅是利用搜索引擎来订阅一些新的资讯。我用google订阅和rss订阅主要是为了获取最新的学术成果。在输入关键词之后，google就会时刻注意这些关键词，当搜索结果出现变化之后，google就会发邮件进行提醒。而RSS订阅主要是用在一些博客上，这些博客专注于某些关键技术的分析，这时候利用RSS订阅和Evernote，当博客进行更新的时候，就会把最新的博文自动创建一篇evernote的笔记。

上面分享了这么多的应用和软件，最后说说我平时是怎么用这些应用，利用碎片的时间来阅读。

每天早晨起床前会刷一下微博，回到实验室打开电脑查邮件的时候就会发现有没有google订阅的新信息，如果有的话，见到扫一下提到的新文章，然后利用zotero把感兴趣的文章抓下来。中午11点时候设置了zaker进行离线下载，然后吃饭回来就可以看一下最新的资讯。晚饭后是书籍的阅读。晚上回到宿舍的路上，利用之前下载好的音频订阅，听一下电视节目。睡前在用网易云阅读来看图褂，了解每天的微博热点。周末会利用电脑看之前抓到pocket的深度文章，然后再整理一下花瓣收集的东西。

这就是我的碎片阅读习惯。自从有效地利用这些工具之后，不再过度地刷微博，同时每天能够获取大量有用的信息。

人来人往

虽说今年的寒假期间有经过广州，但几乎完全没有在广州好好停留，所以前两天回到广州应该算是阔别广州整整一年后的回归。似乎上天也能感受到这次回归的意义，上海和广州的天气给我最贴切的预示。在上海连续几天的高温之后，在我离开的那天居然倾盆大雨；而广州在连日暴雨之后，在我回归当日给了我最晴朗的拥抱。

在广州一切的感觉都是如此熟悉，或许是在上海呆久了，这种熟悉的感觉似乎来得是那么的从容不迫。能够听到熟悉的粤语，能感受到广州那市井的味道，也能浓浓的人情味。

在上海的一段日子，经过前段时间的一系列情感低落，加上广州这座城市对我来说实在是拥有太多太多的熟悉与回忆，以致于还天真地认为广州是我无法触碰的禁区，不情愿，甚至是抗拒回到这个城市，而这一切在我踏上她的土地的那一刻完全被颠覆。

和朋友的聚餐实在是出乎我的意料，我没有想到过会有这么多老朋友会来聚会，也不晓得是大家太久没聚比较渴望聚会，还是因为我回来的面子实在是够大，姑且容许我意淫一下是后者吧（笑）。

餐桌上看着各位喝着我特地跑去买的白葡萄酒，听着各位谈论各自的工作，乐趣与艰辛齐飞，吐槽共感慨一色，这一切比红酒更令人陶醉。忽然发现每个朋友都是携眷出席，心里又是一阵感慨。这班伙伴们，从中学开始认识，每个人有过多多少少的感情经历，我们也共同见证过他们的各自另一半。在聊天的期间，我悄悄打量了下朋友们的伴侣，突然发现似乎那些有过几段感情经历的人，择偶的眼光有所下降。之前娜姐也和我说这个问题，我想一方面是因为一个人经历过一段感情之后，慢慢懂得珍惜身边人，看重的不仅仅是相貌；另一方面是因为失去了足够的耐心，凑合的心理在作怪。

生活的美，原来不曾发现

今天小龙问我，怎么博客没有更新了，才发现原来已经一个月没有更新博客了。

回首想想，这个月发生了挺多事情的，也挺忙的，所以就没空更新博客了。

毕业季

是的，转眼间一年就过去，又到毕业季，可是今年的毕业季并不属于我们。班里的毕业季还组织了周年活动，招呼大家回学校看看。由于还忙着考试，加上距离实在是太远，所有没能抽空回去看看同学，老实说，挺可惜的。后来在微博上看了大家的照片，还有从异国录制的视频，回想起本科四年的感情还是满深厚的。

这个毕业季送走了研三毕业的各位师兄，毕业聚餐上大家都喝多了，在碰杯的时候猛然发现，这些陪伴了我一年的学长即将离开我，心里依依不舍的心情油然而生。刚来实验室时，从一个人都不认识，到渐渐和师兄混熟。每天一起吃饭，休息的时候一起吹牛，晚上一起打球，一起跑步。生日的时候一起聚餐，一起唱K，当然还少不了一起吐槽。在相处的过程中，他们教会我很多东西，无论严谨认真的学术态度，还是乐观向上的生活态度，都让我受益匪浅。和怎么样的人生活在一起，就会有什么样的生活。来复旦给我最深体会，莫过这一简单却难以察觉的道理。

在毕业聚餐上的我，借着冒牌摄影师的名义，逃过好几轮敬酒，在最后相聚的时候依稀保持着清醒的记忆，在师兄们都喝得东倒西歪的时候，背上被遗忘在酒店里的书包，带着一群醉汉过马路，在大街上拦出租车，把他们塞进出租车，然后坐地铁回宿舍。心里默默祝福他们。

关小妹在毕业聚餐上哭着说，你们走了我怎么办，再没有人对我那么好了，再也没有人陪我一起吃饭。的确，或许这一别，以后的相聚要等上更久的日子。可是我突然间想到，他们给我们传递的正能量，要是我们也能继续传承下去，等到我们毕业的时候，也有学妹对我们说，你们走了就再没人都我们这么好了，这该多好呀！

考试后的时光

已经不再像本科那样，考试后就纷纷踏上归途。考完试的日子才是真正能够全心全意干活的日子。前一段时间每天都在苦逼地干活，从开始的一筹莫展到后来慢慢开始有一点曙光，再到后面能够大致能够出来结果，这个过程现在回过头来在体会下，真是别有一番滋味。

在苦逼的干活的日子里，阅读与跑步始终是我最大的慰藉。坚持灵魂与肉体都走在路上。每天利用饭后的闲暇，静静地度过一段阅读时光，从翻动的书页里聆听一个动人的故事，这种感觉实在太美妙了。

天气热了，在操场上跑步变得越来越难熬，简单粗暴地绕着操场跑步的确让人感到厌倦。上周开始尝试路跑，从张衡路绕到金科路，然后在跑到高科中路，有时跑远点到祖冲之路，或者再远点到龙东大道，然后再绕回科苑路，最后跑回宿舍。在路跑途中不断与路灯擦肩而过，皎洁的月光与川流的车流的灯光交错，终于才察觉到原来被忽略的生活的美好。

Sublime Markdown组合成轻量级笔记

Sublime

之前的编辑器一直在使用NotePad++,虽说没什么太多使用的功能，但是觉得用得习惯，也就一直坚持用了下来。最近经Z总介绍，发现了Sublime这个编辑器，试用了大概一周，果断换代NotePad++，改用Sublime。

关于Sublime的简单介绍可以参考Sublime Text 2 入门及技巧

简单来说，吸引我的地方有：

  • 很炫的界面
  • 很好的扩展，插件很多，非常实用
  • 大部分操作都可以使用命令的形式输入

Markdown

Markdown是一种轻量级标记语言，开源社区用得很广。感觉不错是因为它能提供一个简单的笔记排版功能，相对于Word鼠标操作十分繁琐来说，配合Sublime和Vim来用，可以说几乎不用点鼠标，更加专注在文字上，而不是排版上。

当然，Markdown并不是所写即所见的（和LATEX一样），所以一开始会有点不习惯。可是使用一段时间后，觉得很顺手，甚至觉得有点不够用，想去用LATEX。所以感觉LATEX上手很难的话，可以先使用一下Markdown，体验一下这种标记语言的魅力也不错。

Markdown的语法很简单，感兴趣的话，可以看看Markdown 语法说明 (简体中文版)

Sublime+Markdown

要想把Markdown的内容显示出来的，需要在Sublime中装插件。很多人会推荐Markdown Previewer和MarkdownEditing，我试用后觉得都太弱了。如果用这个组合的话，强烈建议使用OmniMakeup Previewer这个插件。

相比于Markdown Previewer，OmniMakeup Previewer不仅能够实时渲染，而且支持多种格式混合使用，比如可以在Markdown中使用LaTeX的数学公式。

比如需要显示 a2+b2=c2 这样的公式，只需要输入$a^2 + b^2 = c^2$就可以了，使用了MathJax来渲染，支持LaTeX支持的所有数学公式

Sublime + Evernot

平时使用Evernote来做笔记，只要在Sublime中装上插件，可以把Markdown的文字渲染好直接发送到Evernote，这个功能真的满不错。只不过是渲染的效果很搓而已。如果需要把Markdown的内容写到笔记去，推荐两个方法：

使用OmniMakeup Previewer渲染，然后用Evernote Clip到Evernote笔记
改用Wiz笔记，最近在使用，感觉很不错
